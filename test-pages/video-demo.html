Frame capture test

When remote detection is enabled, the extension periodically captures a frame from visible videos (same-origin only) and sends a downscaled thumbnail to the configured remote classifier endpoint. Cross-origin videos (e.g., YouTube embeds) cannot be frame-captured due to browser CORS policies.

To test frame capture: host a video on the same origin as this test page, load the page via a local web server (e.g., npx serve test-pages), and enable remote detection in the popup.
